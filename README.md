# 🍜 Ứng Dụng Đồ Ăn Vặt

Ứng dụng đặt món ăn vặt trực tuyến với giao diện đẹp mắt, tối ưu cho laptop, iPad và điện thoại di động.

## ✨ Tính năng

- 🎨 **Giao diện hiện đại**: Thiết kế gradient đẹp mắt, thu hút
- 📱 **Responsive Design**: Tự động điều chỉnh cho mọi thiết bị (laptop, iPad, mobile)
- 🛒 **Giỏ hàng thông minh**: Thêm/xóa món, điều chỉnh số lượng
- 🔍 **Tìm kiếm**: Tìm kiếm món ăn nhanh chóng
- 🏷️ **Nhãn HOT/SALE**: Hiển thị món đặc biệt và khuyến mãi
- 💰 **Tính tổng tự động**: Tự động tính tổng giá trị đơn hàng
- 🎉 **Thông báo**: Hiệu ứng thông báo khi thêm món vào giỏ

## 🚀 Cách sử dụng

1. **Mở file APP.html** bằng trình duyệt web (Chrome, Firefox, Safari, Edge)
2. **Duyệt món ăn**: Xem danh sách các món ăn vặt
3. **Thêm vào giỏ**: Click nút **+** để thêm món vào giỏ hàng
4. **Xem giỏ hàng**: Click icon 🛒 ở góc phải trên cùng
5. **Điều chỉnh số lượng**: Dùng nút +/- trong giỏ hàng
6. **Đặt hàng**: Click nút "Đặt hàng" để hoàn tất

## 📱 Responsive Breakpoints

- **Desktop/Laptop**: > 1024px - Hiển thị lưới 3-4 cột
- **iPad/Tablet**: 768px - 1024px - Hiển thị lưới 2-3 cột
- **Mobile**: < 768px - Hiển thị 1-2 cột
- **Small Mobile**: < 480px - Hiển thị dạng danh sách ngang

## 🎨 Công nghệ sử dụng

- **HTML5**: Cấu trúc trang web
- **CSS3**: Styling với Flexbox, Grid, Animations
- **JavaScript**: Xử lý logic giỏ hàng và tương tác
- **Responsive Design**: Media queries cho mọi thiết bị

## 📂 Cấu trúc file

```
APP_CHIP/
├── APP.html        # File HTML chính
├── style.css       # File CSS styling
├── script.js       # File JavaScript logic
└── README.md       # Hướng dẫn sử dụng
```

## 🎯 Các tính năng chính

### Giỏ hàng
- Thêm món ăn vào giỏ
- Tăng/giảm số lượng
- Xóa món khỏi giỏ
- Tính tổng tự động
- Lưu trữ trong session

### Tìm kiếm
- Tìm kiếm theo tên món ăn
- Lọc kết quả real-time
- Không phân biệt hoa thường

### Responsive
- Tự động điều chỉnh layout
- Tối ưu cho touch screen
- Hiệu ứng mượt mà trên mọi thiết bị

## 💡 Tips

- Nhấn **ESC** để đóng giỏ hàng
- Click vào nền mờ để đóng giỏ hàng
- Sử dụng thanh tìm kiếm để tìm món nhanh
- Hover vào món ăn để xem hiệu ứng zoom

## 🔧 Tùy chỉnh

Bạn có thể dễ dàng tùy chỉnh:
- Thêm món ăn mới trong `APP.html` và `script.js`
- Thay đổi màu sắc trong `style.css`
- Thêm tính năng mới trong `script.js`

## 📞 Hỗ trợ

Nếu gặp vấn đề, hãy kiểm tra:
1. File có đầy đủ 3 files: APP.html, style.css, script.js
2. Các file phải cùng thư mục
3. Trình duyệt hỗ trợ JavaScript
